# Hiroshi Funaoka — Portfolio

Personal portfolio website for Hiroshi Funaoka, Software Developer based in Japan.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Framer Motion
- i18n (EN, JP, ES, PT)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
